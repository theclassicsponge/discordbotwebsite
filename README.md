### How to Start:

1. make a virtual environment: 

(in cmd)
	<pre><code>
  	python -m venv env
	</code></pre>
2. run requirements.txt:

(in cmd)
cd into website:
	<pre><code>
  	C:\Users\<name>\Desktop\discordwebsite\website
	</code></pre>

	<pre><code>
  	pip install -r requirements.txt
	</code></pre>

3. start server:

	<pre><code>
  	python website.py
	</code></pre>

4. open website:

	go to http://127.0.0.1:5000/
